
THIS DIRECTORY CONTAINS ALL RELEVANT MATERIAL FOR THE TRIMMING OF ILLUMINA FASTQ SEQUENCES USING TRIMMOMATIC

-input_data is empty, as the FastQ files are too large to be stored there
-main_output contains the 4 x 2 outputs from trimmomatic, named to indicate the organism, foward/reverse and paired/unpaired.
-scripts contains the script to run trimmomatic for both mutants at the same time. It also contains separate summaries and logfiles for DV3 and HP126.
